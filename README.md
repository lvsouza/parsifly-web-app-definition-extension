# parsifly-web-app-definition-project

## Overview 

This extension allows you to define and specify all about a web app.

## Install

Run yarn to install the dependencies

```
yarn
```

## Run

Run yarn dev to run the extension in `http://localhost:5555/package.json`

```
yarn dev
```

## Build

Run yarn build to build the project

```
yarn build
```

## Version

It is necessary to update version before publish.

## Deploy

Run npm publish to deploy this extension to npm.

```
npm publish
```
